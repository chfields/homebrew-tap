# Homebrew Tap

Personal Homebrew tap for @chfields.

## Installation

```bash
brew tap chfields/tap
```

## Available Casks

### ghostty-theme-picker

Menu bar app to launch [Ghostty](https://ghostty.org) with random themes and manage workstreams. Perfect for running multiple Claude Code sessions with distinct visual identities.

```bash
brew install --cask ghostty-theme-picker
```

**Features:**
- Window Switcher (⌃⌥P) - quickly switch between Ghostty windows
- Claude state detection - windows waiting for input appear at top
- Workstreams - named presets with themes, directories, and commands
- Random Theme - launch with a random theme from 300+ built-in themes

**Requirements:**
- macOS 13 (Ventura) or later
- [Ghostty](https://ghostty.org) terminal emulator
